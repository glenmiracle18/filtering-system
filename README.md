## :toolbox: Getting Started

### :bangbang: Prerequisites

- Clone this repo
```bash
git clone https://github.com/glenmiracle18/filtering-system
```
- rename the dotenv file
```bash
cp .env.example .env
```
- run locally
```bash
yarn dev
```


### :gear: Installation

Clone the repo
```bash
git clone https://github.com/glenmiracle18/filtering-system
```
install all dependencies
```bash
pnpm install
```
